Detailed Description

Console process structure.

A console process has a channel input it reads data from, it then displays it on the standard output (e.g : console). User can choose a prefix and a suffix to respectively prepend and append to the displayed message. IMPORTANT : prefix and suffix must have the '\0' terminating character.
