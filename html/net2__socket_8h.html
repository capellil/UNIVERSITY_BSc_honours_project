Creates a socket and binds it to the given port. Uses TCP/IP protocols.

Parameters
net2_socketA pointer to store the created net2 socket in.
portThe port to be used.
Returns
  • SUCCESS : 0.
  • FAILED :
    • -1 : socket creation failed.
    • -2 : socket SOL_REUSEADDR option set failed.
    • -3 : socket binding failed when trying with given port.
    • -4 : the socket information extraction failed.
Precondition
net2_socket points to a non-NULL net2_socket.

Parameters

dataA pointer on the data allocated space.
data_lengthThe number of bytes to read from the socket.
Returns

  • FAILED
