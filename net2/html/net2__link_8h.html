int net2_create_link_rx ( struct net2_link_rx_t *  net2_link_rx,
unsigned short  port 
)

Creates a link RX by creating its socket and binding it to the given port. Does not start listening nor accepting process.

Parameters
net2_link_rxA pointer to an already allocated memory area to store in the created link RX.
portThe port to listen and accept on.
Returns
  • SUCCESS : 0.
  • FAILURE : Negative number, errno is set appropriately.
Precondition
net2_link_rx points to an already allocated memory area.
Postcondition
  • SUCCESS :
    • net2_link_rx->_net2_socket created and binded to the port "port".
    • net2_link_rx->_partner_socket = NULL
    • net2_link_rx->read = &net2_read_from_link_rx.
  • FAILURE : net2_link_rx unchanged.
