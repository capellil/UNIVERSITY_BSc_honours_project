enum net2_channel_type_e *  net2_channel_type,
unsigned int  channel_number 
)

Searches the channel corresponding to the given channel number.

If the channel is NOT found, the net2_channel_generic pointer is set to NULL, otherwise it is set to the address of the found channel.

Parameters
net2_channel_genericA pointer on a pointer on a not allocated area to store the address of the found channel. MUST BE "void**" because it cannot be determined if the found channel will be of type input or output.
net2_channel_typeA pointer on a channel type to store the found channel type (CHANNEL_INPUT or CHANNEL_OUTPUT).
channel_numberThe channel number to search.
Returns
  • SUCCESS : 0
  • FAILURE : -1
